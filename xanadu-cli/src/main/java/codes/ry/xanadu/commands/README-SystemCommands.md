# System Commands

System and utility commands for Xanadu.

## Commands

### Log Level
- `loglevel` - Shows the current log level
- `loglevel <level>` - Sets the log level (error, warn, info, debug)

### Provider Management
- `enable <provider>` - Enables a command provider
- `disable <provider>` - Disables a command provider

### Help
- `help` - Lists all available commands
- `help <command>` - Shows detailed help for a command

### Other
- `quit` or `q` - Exits the REPL
- `lastexception` - Shows the last exception stack trace
