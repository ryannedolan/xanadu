# Draw Commands

Commands for drawing graphics using a canvas and turtle graphics.

## Commands

### Canvas Commands
- `draw clear` - Clears the drawing buffer
- `draw point <row> <col>` - Draws a point at coordinates
- `draw line <row1> <col1> <row2> <col2>` - Draws a line
- `draw rect <top> <left> <height> <width>` - Draws a rectangle
- `draw text <row> <col> <text...>` - Draws text at position
- `draw show <height> <width>` - Renders the drawing buffer

### Turtle Graphics Commands
- `draw pen up` - Lifts the pen (stops drawing)
- `draw pen down` - Lowers the pen (starts drawing)
- `draw move <row> <col>` - Moves turtle to position
- `draw forward <distance>` - Moves turtle forward
- `draw left <degrees>` - Turns turtle left
- `draw right <degrees>` - Turns turtle right
- `draw heading <degrees>` - Sets turtle heading
- `draw home` - Returns turtle to origin
