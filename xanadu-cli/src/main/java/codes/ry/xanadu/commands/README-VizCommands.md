# Visualization Commands

Commands for creating charts and visualizations in the terminal.

## Commands

- `bar <values...>` - Creates a vertical bar chart
- `hbar <values...>` - Creates a horizontal bar chart
- `spark <values...>` - Creates a sparkline chart
