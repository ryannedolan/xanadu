# Visualization Commands

This document demonstrates the visualization commands for creating charts and graphs.

## Sparkline Command

The `spark` command creates a sparkline chart from a series of numbers:

```xanadu
> spark 1 2 4 8 16 32 64
▁▁▁▂▃▅█
```

## Bar Chart Command

The `bar` command creates a vertical bar chart:

```xanadu
> bar 10 20 30
...
```

More examples with different values:

```xanadu
> bar 5 10 15 20 25
...
```

## Horizontal Bar Chart Command

The `hbar` command creates a horizontal bar chart:

```xanadu
> hbar 10 20 30
...
```

Another example:

```xanadu
> hbar 25 50 75 100
...
```
